# Game Logic

## Scope
- The application contains a browser Pong-style game component.

## Core Rules
- Ball moves continuously and bounces from top and bottom boundaries.
- Player controls paddle movement.
- Score changes based on successful returns and misses.
- On terminal condition, game state resets according to component rules.

## Sync Policy
- Update this file in the same task when gameplay behavior, scoring, controls, or state transitions change.
